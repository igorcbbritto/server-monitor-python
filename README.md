# Server Monitor Python

Projeto em Python para monitoramento de recursos do sistema, exibindo uso de CPU, memória RAM e disco com alertas básicos.

## Objetivo

Este projeto foi desenvolvido para simular uma rotina simples de monitoramento de servidor, permitindo acompanhar o consumo de recursos do sistema e identificar situações de uso elevado.

## Funcionalidades

- Verificação de uso de CPU
- Verificação de uso de memória RAM
- Verificação de uso de disco
- Alertas quando o consumo ultrapassa limites definidos

## Tecnologias utilizadas

- Python 3
- psutil
